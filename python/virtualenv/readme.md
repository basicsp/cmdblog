## 不带任何依赖的安装虚拟环境（指定python3）

virtualenv --no-site-packages venv-name -p python3

## 激活虚拟环境

source venv/bin/activate

## 退出虚拟环境

deactivate

## 删除虚拟环境

rmvirtualenv venv
